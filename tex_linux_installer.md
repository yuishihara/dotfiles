# How to intsll TexLive in Ubuntu

```sh
$ sudo -S apt-get install texlive-lang-japanese
$ sudo -S apt-get update
$ sudo -S apt-get upgrade
$ sudo -S apt-get install texlive-lang-cjk
$ sudo -S apt-get install texlive-fonts-recommended
$ sudo -S apt-get install texlive-fonts-extra
$ sudo -S apt-get install xdvik-ja
$ sudo -S apt-get install dvipsk-ja
$ sudo -S apt-get install gv
$ kanji-config-updmap-sys ipaex
```
